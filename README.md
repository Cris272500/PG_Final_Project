<div style="text-align: center;"> <h2>Resumen del proyecto</h2> </div>
Nuestro equipo ha desarrollado un entorno virtual 3D que permite a los usuarios sumergirse en un campo de fútbol, específicamente en el estadio del FC Barcelona, ubicado en España, Europa. Con este proyecto, buscamos superar las barreras de distancia y económicas, permitiendo que cualquier aficionado de este equipo pueda experimentar de manera inmersiva el ambiente del estadio del FC Barcelona.

<div style="text-align: center;"> <h2>Tecnologías utilizadas</h2> </div>
Para el desarrollo de este proyecto, hemos empleado una variedad de tecnologías, incluyendo lenguajes de programación, control de versiones y software de modelado 3D.

1. **Python**: Elegimos Python como el lenguaje de programación principal para el desarrollo del proyecto. Utilizamos módulos como **moderngl, pygame, PYWAVE y pyopengl**.

	- **Moderngl:** Para el renderizado moderno de gráficos 3D utilizando opengl.

	- **Pygame**: Para el manejo de eventos y la integración de elementos multimedia.

	- **PYWAVE:** Para la simulación de ondas y efectos sonoros en el entorno virtual

	- **Pyopengl:** Para la integración de gráficos 3D utilizando opengl.

2. **Blender:** Blender fue utilizado para probar y modificar los modelos 3D.

3. **Git**: Utilizamos Git para el control de versiones y para facilitar el desarrollo colaborativo del proyecto.

<div style="text-align: center;"> <h2>Project Summary
</h2> </div>
Our team has developed a 3D virtual environment that allows users to immerse themselves in a soccer field, specifically the FC Barcelona stadium located in Spain, Europe. With this project, we aim to overcome distance and economic barriers, enabling any fan of this team to experience the immersive atmosphere of the FC Barcelona stadium.

<div style="text-align: center;"> <h2>Technologies Used</h2> </div>
For the development of this project, we have used a variety of technologies, including programming languages, version control, and 3D modeling software.

1. **Python**: We chose Python as the main programming language for the project development. We used modules such as moderngl, pygame, PYWAVE, and pyopengl.
	- **Moderngl:** For modern 3D graphics rendering using OpenGL.

	- **Pygame**: For event handling and integration of multimedia elements.

	- **PYWAVE:** For wave simulation and sound effects in the virtual environment.

	- **Pyopengl:** For integrating 3D graphics using OpenGL.

2. **Blender:** Blender was used for testing and modifying the 3D models.

3. **Git**: We used Git for version control and to facilitate collaborative development of the project.

<div style="text-align: center;"> <h2>Installations</h2> </div>

Previously, we explained the different modules we used in this project. To use these modules, we will perform the following configurations, before performing the installations, we need to clone the repository using:
`git clone https://github.com/Cris272500/PG_Final_Project.git` once this is done, we will proceed with the installations.

1. Install Python
	To verify that Python is installed correctly, open the command prompt (CMD) and execute the following command:
	**`python --version`**
	The terminal should display the installed version of Python. If Python is not installed, the project cannot be executed. For more information on installation, you can follow the tutorial that explains how to install it.
	<div align="center">
        <a href="https://youtu.be/yivyNCtVVDk?si=3PrPei8bh1wLwRFT">
            <img src="https://img.youtube.com/vi/yivyNCtVVDk/0.jpg" alt="Video de YouTube" style="width:560px;height:315px;">
        </a>
    </div>

---
2. Create a Virtual Environment
	To use the modules used in this project, it's necessary to create a virtual environment so that they can be installed correctly with their respective versions. To do this, we use:
	**`python -m venv env`**
	This command will create a folder named `env` that will contain all the necessary modules for this project.
---
3. Activate the Virtual Environment
	Once created, we will execute the following command to activate it and install the modules:
	`.\env\Scripts\activate`
	Afterwards, to install all the modules used in the project with their respective versions, use the command:
	`pip install -r requirements.txt`
---
4. Running the Project
	Once we have completed the previous steps, execute `cd Project-Origin-Version 1.0`, and within this folder, you should see a structure like this:
	![image](https://github.com/Cris272500/PG_Final_Project/assets/113935131/b1e65d15-9e6a-4e4e-a211-bf9d1429c4cb)

	Then, execute `python UI.py`. This way, we will be running the project.
